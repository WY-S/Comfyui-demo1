# Comfyui-Wenyi-demo1 · HappyHorse 视频生成节点

基于阿里百炼 **HappyHorse** 视频生成 API 的 ComfyUI 自定义节点集。支持在 ComfyUI 画布中一键调用文生视频 / 图生视频 / 参考生视频 / 视频编辑能力，并在节点内直接内嵌播放生成结果。

<font color="red">注意：代码仅供演示使用</font>

## 前置条件

1.  需要有百炼API key
    
    [https://bailian.console.aliyun.com/?tab=model#/api-key](https://bailian.console.aliyun.com/?tab=model#/api-key)
    
2.  如果需要测试 图生视频 / 参考生视频 / 视频编辑 需要有阿里云OSS，并获取AK，SK。
    

获取AK/SK：[https://help.aliyun.com/zh/ram/product-overview/quick-start-create-and-use-accesskey-pairs-for-programmatic-calls](https://help.aliyun.com/zh/ram/product-overview/quick-start-create-and-use-accesskey-pairs-for-programmatic-calls)

阿里云OSS：[https://help.aliyun.com/zh/oss/user-guide/console-quick-start](https://help.aliyun.com/zh/oss/user-guide/console-quick-start)

## 3、启动ComfyUI

如果电脑有GPU，可以选择run\_nvidia\_gpu.bat。如果没有GPU，可以用CPU启动，启动run\_cpu.bat。

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqreb6Z5p0enak/img/38e19504-4844-4fd7-85e1-4669db4fb6b0.png)

启动后会自动打开浏览器：[http://127.0.0.1:8188/](http://127.0.0.1:8188/)

## 4、在 ComfyUI 画布中测试Happyhorse模型

1.  测试用的workflows在\example\_workflows目录下。用鼠标拖到浏览器页面即可。
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqreb6Z5p0enak/img/0a1a26d6-9825-438e-97e8-3d64903766f8.png)
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqreb6Z5p0enak/img/65dc1b91-c295-4d6f-902c-12e659952e48.png)
    
2.  安装python依赖包。
    

因为ComfyUI使用的是内嵌的 Python（python\_embeded），需要在对应的环境里安装pip install oss2

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqreb6Z5p0enak/img/9f5e3435-8678-4d9b-9346-62db08a7eeed.png)

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqreb6Z5p0enak/img/e09efedf-e9a2-4209-a75e-c859294008d5.png)

3. 开始测试

在node中分别填入

*   百炼api\_key
    
*   AK
    
*   SK
    
*   OSS Bucket 名称：可以在控制台bucket列表找到
    
*   OSS 区域域名：阿里云OSS所在的地区，如 oss-cn-beijing.aliyuncs.com（必须与 bucket 所在区域一致）
    

**注意：填入的信息会被保存到config.json文件中，因此只需在第一次使用时填写即可，后续测试时不需要再次填写（直接执行即可）。如果需要修改api key，可以再次填写或到config.json文件中直接修改。**

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqreb6Z5p0enak/img/f4604906-a20e-4786-b694-d3af9b2766da.png)

## 5、素材目录

导入的图片会保存在：\ComfyUI\_windows\_portable\ComfyUI\input目录

导入的图片会上传到阿里云OSS happyhorse目录

生成的视频除了在网页中显示外，保存到：\ComfyUI\_windows\_portable\ComfyUI\output\happyhorse目录
